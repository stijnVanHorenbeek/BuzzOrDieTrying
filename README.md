# BuzzOrDieTrying

This project is a Proof of Concept (PoC) demonstrating Masstransit request-response messaging with proper error handling.

## Requirements

- [Docker](https://www.docker.com/)

## Getting Started

To run the project, use Docker Compose:

```sh
docker compose up --build
```

This will build and start all necessary services.

## Features

- Masstransit request-response pattern
- Robust error handling
